# Guia de Início Rápido

## Para Desenvolvedores

### Pré-requisitos
- Node.js 18+ 
- npm ou yarn
- Git

### Instalação

1. **Clone o repositório**
   ```bash
   git clone https://github.com/seu-usuario/curriculo-rapido.git
   cd curriculo-rapido
   ```

2. **Instale as dependências**
   ```bash
   npm install
   ```

3. **Execute em desenvolvimento**
   ```bash
   npm run dev
   ```
   Acesse `http://localhost:5173`

4. **Build para produção**
   ```bash
   npm run build
   ```
   Os arquivos serão gerados em `dist/`

5. **Preview do build**
   ```bash
   npm run preview
   ```

## Para Usuários

### Uso Online
1. Acesse [curriculorapido.com.br](https://curriculorapido.com.br)
2. Preencha seus dados no formulário
3. Escolha um modelo de currículo
4. Visualize o preview em tempo real
5. Baixe em PDF quando estiver pronto

### Funcionalidades Principais

#### Criar Currículo
- Preencha as seções: Dados Pessoais, Experiência, Formação, Habilidades
- Use o editor de texto rico para formatação
- Adicione uma foto profissional (opcional)
- Escolha um modelo que combine com você

#### Verificar ATS
- Role até a seção "Verificador ATS"
- Clique em "Verificar Currículo"
- Veja o score e sugestões de melhoria

#### Sugestões para Vagas
- Cole a descrição completa da vaga
- Clique em "Analisar Vaga"
- Veja o score de compatibilidade e palavras-chave

#### Salvar e Gerenciar
- Clique no botão "Gerenciar Currículos"
- Salve múltiplos currículos
- Renomeie, duplique ou exclua quando necessário

#### Exportar
- Use o menu de exportação (ícone de três pontos)
- Escolha: PDF, Word, JSON ou Imprimir
- Para backup, use JSON

#### Compartilhar
- Clique em "Compartilhar"
- Copie o link ou compartilhe nas redes sociais
- O link contém todos os dados do currículo

## Atalhos de Teclado

- `Ctrl + S` - Salvar currículo
- `Ctrl + P` - Imprimir/PDF
- `Ctrl + /` - Ver atalhos
- `Esc` - Fechar modais

## Dicas

### Para Melhor ATS Score
- Use palavras-chave da vaga
- Mantenha formatação simples
- Evite gráficos e tabelas complexas
- Inclua resumo profissional

### Para Melhor Visual
- Escolha modelo adequado à sua área
- Use foto profissional (opcional)
- Mantenha consistência nas datas
- Revise ortografia

### Para Múltiplas Vagas
- Crie versões diferentes do currículo
- Salve com nomes descritivos
- Adapte resumo e habilidades para cada vaga
- Use sugestões de vagas para otimizar

## Solução de Problemas

### PDF não gera
- Verifique se há conteúdo no currículo
- Tente em outro navegador
- Limpe o cache do navegador

### Dados não salvam
- Verifique se localStorage está habilitado
- Limpe dados antigos se necessário
- Tente em modo anônimo

### Layout quebrado
- Atualize a página (Ctrl + F5)
- Limpe cache do navegador
- Verifique se JavaScript está habilitado
